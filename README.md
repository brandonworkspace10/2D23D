# Pipeline Studio (Frontend)

Single-page, mocked Pipeline Studio for the 2D-to-3D product conversion platform.

## Requirements
- Node.js 18+ (not installed in this environment)
- npm or pnpm

## Install (later)
```bash
# npm
npm install
npm run dev

# pnpm
pnpm install
pnpm dev
```
